Diseño Y Programacion De Paginas Web

Este documento presenta una introducción al diseño y programación de páginas web. Explica los conceptos básicos necesarios para publicar una página web, incluyendo el lenguaje HTML, editores de páginas web, alojamiento web y más. Luego procede a detallar 16 capítulos que cubren temas como diseño web, programación con lenguajes como HTML, JavaScript, PHP y más, así como temas relacionados como bases de datos y CSS.

Web Hacking 101 en Español - Cómo hacer dinero hackeando éticamente

El libro cubre varias técnicas comunes de hacking como inyección de HTML, contaminación de parámetros HTTP, inyección de CRLF, falsificación de solicitudes entre sitios, vulnerabilidades en la lógica de aplicaciones, ataques de script entre sitios, inyección SQL y más. Los autores esperan que este libro ayude a las personas a aprender sobre seguridad web de una manera ética y lucrativa
